時代劇の**『初恋のジャスミン (莫离)』**は、驚くほど良い中国語の教師です：同じ宮廷語彙が毎エピソードで繰り返されるので、意識して見るだけで早く覚えられます。ここにツールキットがあります — 白鹿の復讐ロマンスで常に耳にする言葉と、実際に策略が働くいくつかの成語です。これらを学べば、字幕がオプションのように感じ始めます。

この番組に初めてですか？[キャスト、プロット、視聴ガイド](/blog/the-first-jasmine-cast-plot-where-to-watch-bai-lu)から始めましょう。

## 宮廷と地位の語彙

| 用語 | ピンイン | 意味 |
|---|---|---|
| 王爷 | wáng yé | 王子 / 皇帝の主（莫修耀に対する呼称） |
| 定王 | dìng wáng | 「定王」 — 莫修耀の称号 |
| 圣旨 | shèng zhǐ | 皇帝の詔勅 / 命令 |
| 赐婚 | cì hūn | **皇帝の命令によって授けられた結婚** — 主人公を結びつける正確な出来事 |
| 嫡女 | dí nǚ | 正妻から生まれた娘 — 葉莉の地位 |
| 世家 | shì jiā | 貴族の家系 / 大家族 |
| 太后 | tài hòu | 皇太后 |
| 太妃 | tài fēi | 皇太妃 |
| 战神 | zhàn shén | 「戦神」 — 莫修耀の以前の称号 |
| 篡位 | cuàn wèi | 王位を奪う |

## テーマとプロットの語彙

| 用語 | ピンイン | 意味 |
|---|---|---|
| 复仇 | fù chóu | 復讐 / 復讐心 — プロットのエンジン |
| 权谋 | quán móu | 政治的策略 / 宮廷の陰謀（ジャンルのタグ） |
| 先婚后爱 | xiān hūn hòu ài | 「先に結婚し、後に愛する」 — ロマンスのトロープ |
| 双强 | shuāng qiáng | 「二人の強者」 — 両方の主人公が有能で、一方が他方を救うのではない |
| 书院 | shū yuàn | 古典アカデミー / 儒教学校 |

**学習のヒント：** 言葉の壁を暗記しようとしないでください。上記から5つを選び、最初の2エピソードで*それらを聞く*ようにしましょう。膝をついた宮廷で叫ばれる圣旨 (shèngzhǐ) や、主人公に対して使われる王爷 (wángye) を聞くことで、その言葉がフラッシュカードよりもずっとよく定着します。時代劇はこのレジスターを無限に繰り返します — その繰り返しがあなたの味方です。

## プロットが依存する4つの成語

単語を超えて、これらの四字熟語に注目してください — それらはキャラクターが実際にどのように動くかを説明します：

### 知己知彼 (zhī jǐ zhī bǐ) — 「自分を知り、敵を知る」
孫子の*兵法*から：両方の側を理解すれば、危険にさらされることはありません。誰もが他の誰かを見極めようとしているドラマでは、[知己知彼](/blog/zhi-ji-zhi-bi)は生存の鍵です。

### 运筹帷幄 (yùn chóu wéi wò) — 「指揮幕の中で戦略を立てる」
舞台裏から計画し指揮すること — 戦う前に戦いを勝ち取る。真の権力者の印です。[运筹帷幄](/blog/yun-chou-wei-wo)。

### 深谋远虑 (shēn móu yuǎn lǜ) — 「深い計画、遠くを見据えた思考」
多くの手を先読みすること — 生き残る策略家とそうでない者を分ける資質です。深谋远虑。

### 将计就计 (jiāng jì jiù jì) — 「敵の計画を逆手に取る」
誰かがあなたのために仕掛けた罠を利用して、*彼ら*を打ち負かすこと。復讐プロットで最も満足のいく動きです。[将计就计](/blog/jiang-ji-jiu-ji)。

## 番組を学びの教材として使う方法

1. **物語のためにエピソードを一度見て、言語のためにもう一度見てください。** 2回目は、上記の宮廷用語で一時停止します。
2. **エピソードごとに5語のリストを作成します。** 小さく繰り返すことが、大きく忘れられることよりも効果的です。
3. **成語に注目してください。** キャラクターがシーンで「勝つ」時、彼らがどの成語を使ったのかを尋ねてみてください — それはあなたの耳と戦略的な読みを鍛えます。

復讐プロットの背後にあるより深い成語セットについては、[『初恋のジャスミン』 — 復讐と宮廷陰謀のための中国成語](/blog/the-first-jasmine-chinese-idioms-revenge-palace-intrigue)を参照し、名前を理解するためには、[莫离の意味：タイトルとキャラクター名の説明](/blog/the-first-jasmine-title-character-names-meaning)をご覧ください。
